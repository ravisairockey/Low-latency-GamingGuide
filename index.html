Maximize Your Network Performance

This guide provides the optimal Network Interface Card (NIC) settings for gaming and streaming simultaneously at 1080p resolution. Proper configuration can reduce latency, prevent packet loss, and provide a smoother experience.

Recommended NIC Settings

Property Recommended Setting Reason
Advanced EEE Disabled Reduces latency spikes caused by power saving.
ARP Offload Disabled Small power save; disable for lowest latency.
Energy-Efficient Ethernet Disabled Prevents slow wake-ups from idle state.
Flow Control Enabled Prevents packet loss during congestion.
Gigabit Lite Disabled Avoids auto-downshifting bandwidth.
Green Ethernet Disabled May reduce link speed and add delay.
Interrupt Moderation Disabled Lowers latency by reducing interrupt delay (great for gaming).
IPv4 Checksum Offload Rx & Tx Enabled Reduces CPU usage; doesn't affect latency.
Jumbo Frame Disabled (unless LAN use) Not used by streaming/gaming sites; only helps in local file transfers.
Large Send Offload v2 (IPv4/6) Enabled Improves performance by offloading segmentation.
NS Offload Disabled Unused for most gaming/streaming, saves a tiny bit of power.
Power Saving Mode Disabled Prevents NIC from going to low-power state during streaming/gaming.
Priority & VLAN Disabled Not needed unless your router supports QoS VLAN tagging (rare at home).
Receive Buffers / Transmit Buffers Default or 512+ Higher = better throughput (leave default or increase slightly if adjustable).
Recv Segment Coalescing (IPv4/6) Enabled Reduces CPU interrupts, no real impact on latency.
Shutdown Wake-On-LAN Disabled Not used unless waking up PC remotely.
Speed & Duplex Auto Negotiation Avoid mismatches; don't force 100/1000 unless troubleshooting.
TCP/UDP Checksum Offload (IPv4/6) Enabled Offloads CPU without affecting latency.
Wake on Magic Packet Disabled Not needed unless you're using Wake-on-LAN.
Wake on Pattern Match Disabled Security risk and not used for gaming.
WOL & Shutdown Link Speed Disabled Not needed; avoid low-speed wake issues.
